# Security Policy

This public profile repository is operated by Johnny Watters (`0ai-Cyberviser`) as part of the `0AI` company portfolio.

If you discover a security issue affecting this repository, or a security issue in an associated `0AI` project that should not be disclosed publicly yet, do not open a public issue first.

Report privately to:

- 0ai@cyberviserai.com
- cyberviser@proton.me

Include:

- affected repository
- impact summary
- reproduction details
- any proposed remediation
